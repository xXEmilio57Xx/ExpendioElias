# ExpendioElias

Como primer paso, correr los scripts de la base de datos segun su numero de orden en SqlServer.
Para la conexion a la base de datos solo cambiar el server en la cadena de conexion que se encuentra en la clase 
"Conexion" ubicada en la carpeta "Clases". El tipo de autenticacion por defecto es la Autenticacion de Windows,
de igual manera si se desea cambiar esto es en el mismo apartado.
